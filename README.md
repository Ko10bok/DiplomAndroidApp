# Приложение по контролю качества масла

## Описание
Это приложение предназначено для контроля качества моторного масла. Пользователь (то есть ты, дорогой читатель) вводит основные параметры, а данное приложение сравнивает их с установленными нормативами и выдает результат — пригодно ли масло для использования.  
Данная версия является зачатком моего диплома, тут нет математики, но вложена душа, силы, возможно обиды и страдания.

## Возможности
- Ввод параметров масла
- Сравнение с нормативами
- Оценка пригодности масла

## Технологии
Написана в AndroidStudio на языке Java, при необходимости будет добавлен Kotlin, но пока не добавляю, т.к. не хочу.

## Использование
1. Введите параметры масла в соответствующие поля.
2. Нажмите кнопку проверки.
3. Получите результат — пригодно/непригодно.

## Преписка
Также в приложении данной версии реализована БД, в которой хранятся наши записи. Поэтому, дорогой друг, если ты хочешь, то можешь открыть и посмотреть, что ты там намерялл в прошлом месяце, или удалить, если оно тебе и не надо вовсе.
